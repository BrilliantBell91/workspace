# workspace

Claude Code 실습 프로젝트 모음입니다.

## 프로젝트 목록

### bucket-list-main
버킷 리스트 웹 애플리케이션. Tailwind CSS와 Vanilla JS로 구현된 CRUD 앱이며, Playwright E2E 테스트가 포함되어 있습니다.

- **기술 스택**: HTML, CSS, JavaScript, Tailwind CSS, Playwright
- **주요 기능**: 항목 추가/수정/삭제, 완료 체크, 필터링, LocalStorage 저장

### my-profile-site
개인 프로필 소개 사이트입니다.

- **기술 스택**: HTML, Tailwind CSS

### output-style-test
Claude Code 출력 스타일 실습용 HTML 예제 모음입니다.

- **포함 파일**: 계산기, 카운터, 스톱워치, Todo 앱

### starterkit
인증이 포함된 Next.js 풀스택 스타터 템플릿입니다.

- **기술 스택**: Next.js, TypeScript, Prisma, NextAuth, Tailwind CSS, shadcn/ui
- **주요 기능**: 이메일/비밀번호 로그인, SQLite DB, 보호된 라우팅

### claude-code-mastery
Claude Code 터미널 기본 동작 실습 프로젝트입니다.

### user-memory-setup
Claude Code 사용자 메모리 설정 실습 프로젝트입니다. React Counter 컴포넌트 예제가 포함되어 있습니다.
